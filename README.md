# Age Calculator

This is a simple age calculator project.

It takes a user's date of birth as input and calculates the exact age in years, months, and days.

## How it works

- Enter your date of birth
- Click on calculate
- The program shows your age in years, months, and days

## Example

Input:
04/28/2007

Output:
18 years, 11 months and 17 days
